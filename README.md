# The Charter House

A personal workspace for ideas, experiments, and products.

## Structure

- `ideas/` — raw ideas, one file per idea
- `experiments/` — quick prototypes and tests, one folder per experiment
- `products/` — things being built seriously, one folder per product
- `prompts/` — reusable prompts that work well
- `docs/` — notes, research, and reference material

## Key files

- `LESSONS.md` — what I've learned along the way
- `PROJECT_STATUS.md` — what's active right now
